SEP11 Freedom Project

This project was about creating anything you wanted using Javascript and a seperate component that you would have to learn how to use throughout the year. What makes this project different from the previous freedom project that I have done is that I decided to do this with a partner, Brianna.Together, we decided to create a Spanish Platformer game using a Javascript programming libarary named kaboom.js. Similar to last time, we would both have to present our project to the class and near the end of the school year at an expo.

Along the way there were many challenges that I learned from. One of them having to do with the process of learning the tool we chose to help me make our freedom project game. One of the biggest challenges for both of us was finding the time to plan out the project. We were creating what was called an MVP(Minimum Viable Product) for our project so that we could have a baseline for when we actually start to develop it into a real working game. However, during the process my partner had been out of state on a vacation. Which was a issue because we both worked indivdually on two seperate components of the game that would complete our MVP, my part being to work on the enemies, my partner's part being to intergrate spanish related questions into the game.
